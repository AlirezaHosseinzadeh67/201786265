TO USE AN ALREADY CREATED CONDA VIRTUAL ENVIRONMENT:
- Open Anaconda Prompt
- Open the directory, by typing> cd "directory path"

- You should be able to see (GEOG5995_Final_201786265_env) in front of the next line.
